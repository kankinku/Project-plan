## 본문

투자해서 한 달에 5%만 벌면 돼. 그러니까 과도하게 매일매일 매매할 필요 없어. 필요한 날에만 1,2개만 집중 투자하고 딱 벌고 나와.알았지, 탄지로?​​1억 투자 시, 매달 5% 수익을 차익실현하여 재투자할 경우 월별 누적 원금 변화원금 1억 → 1년 뒤 약 1억 7,958만 원 (≈ 1억 8천만 원)원금 1천만 원 → 1년 뒤 약 1,795만 원 (≈ 1,800만 원)​

​1억 투자 시, 매달 5% 수익률 기준으로 1~50년

​​1억 투자 시, 매달 10% 수익을 차익실현하여 재투자할 경우 연별 누적 변화​💡 다른 투자금 적용 방법:10억 투자자 → 숫자 끝자리 0 하나 추가하기 천만 원 투자자 → 숫자 끝자리 0 하나 빼기 백만 원 투자자 → 숫자 끝자리 0 두 개 빼기 ​

숫자 단위:1억(億) = 10⁸1조(兆) = 10¹²1경(京) = 10¹⁶1해(垓) = 10²⁰1자(𥝱) = 10²⁴1양(穰) = 10²⁸1구(溝) = 10³²​​

​1억 총자산으로 시작, 매달 총자산의 50%만 투자투자금 10% 수익, 수익 절반 재투자,비축현금 = 총자산 절반 + 투자 수익 절반 방식​
